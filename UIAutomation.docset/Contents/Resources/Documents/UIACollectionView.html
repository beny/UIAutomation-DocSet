UIACollectionView Class Reference

Inherits from
UIAScrollView
Availability
Available in iOS 6.0 and later

Overview

The UIACollectionView class allows access to, and control of, elements within a collection view in your app.

For an explanation of how to use this class and related classes, see “Automating UI Testing” in Instruments User Guide.

Inherited Methods

Table 1 provides a list of methods inherited from UIAScrollView.

Table 1  Methods inherited from UIAScrollView

Method

Description

scrollDown

Scrolls down within the specified collection view.

scrollLeft

Scrolls left within the specified collection view.

scrollRight

Scrolls right within the specified collection view.

scrollToElementWithName

Scrolls within the collection view until the named element is displayed on the screen.

scrollToElementWithPredicate

Scrolls within the collection view until the matching element is displayed on the screen.

scrollToElementWithValueForKey

Scrolls within the collection view until the element with the specified value for the specified key is displayed on the screen.

scrollUp

Scrolls up within the specified collection view.
