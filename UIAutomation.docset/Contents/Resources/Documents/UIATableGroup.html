UIATableGroup Class Reference

Inherits from
UIAElement
Availability
Available in iOS 4.0 and later.

Overview

The UIATableGroup class allows access to, and control of, group elements within a table view.

For an explanation of how to use this class and related classes, see “Automating UI Testing” in Instruments User Guide.
